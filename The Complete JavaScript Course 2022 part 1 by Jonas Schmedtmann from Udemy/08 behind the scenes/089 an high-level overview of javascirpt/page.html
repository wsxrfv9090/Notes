High-Level:
        Low-Level languages: Munually Manage All the Recources from CPU and memory.
        High-Level languages: These languages have so called abstractions that can take away those works.
 
  Down side is that these high-level languages such as Python and JavaScript will never be as fast and as optimized as low-level languages such as C.



Garbage-Collection:
  This is a algorithms that belongs to one of the functions that were in high-level languages that can help developers to aside the memory management and do the main coding.
  It automatically removes old and unused objects from the computer memory, in order not to clog it up with unnessasary stuff.
  It automatically cleans the memory from time to time so we don't have to do it manually in our code.
